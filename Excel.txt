Ctrl + espaço -> Seleciona coluna
Shift + espaço -> Seleciona linha
Ctrl + '+' ou Ctrl + Shift + '+' -> Insere linha
Ctrl + '-' ou Ctrl + Shift + '-' -> Exclui linha
Shift + seta -> Seleciona as células na direção da seta
Ctrl + seta -> Seleciona célula da extremidade indicada pela seta
Ctrl + Home -> Seleciona célula inicial (A1)
Ctrl + Shift + seta -> Seleciona todas as células na direção da seta
Ctrl + * ou Ctrl + Shift + 8 -> Seleciona subplanilha
 
TAB -> Quando apertado  vai para a casa ao lado
ENTER -> Vai para a linha de baixo

CTRL + T -> Seleciona a tabela
CTRL + Q -> Analise rapida

Ctrl + Shift + 1 -> Número
Ctrl + Shift + 2 -> Hora
Ctrl + Shift + 3 -> Data
Ctrl + Shift + 4 -> Moeda
Ctrl + Shift + 5 -> Porcentagem
